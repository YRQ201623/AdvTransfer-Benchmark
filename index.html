AdvTransfer Benchmark

A Comprehensive Framework for Fair Evaluation of Adversarial Transferability Methods

📋 Overview

The AdvTransfer Benchmark provides a rigorous and fair evaluation framework for adversarial transferability methods. Unlike existing evaluations that use inconsistent model architectures and preprocessing pipelines, our benchmark establishes unified conditions where all attack methods compete on equal footing.

This platform implements representative transferability methods including NAA, COPIA, and their variants with a unified codebase, ensuring fair comparison across different algorithmic innovations. The benchmark systematically removes advantages from model ensemble and unlimited query budgets, providing a transparent baseline for evaluating black-box adversarial attacks.

🌟 Key Features
🎯
Unified Architecture
All methods are evaluated on identical network structures, eliminating performance differences arising from architectural advantages rather than training procedures.
⚙️
Standardized Settings
Consistent training schedules, optimizers, learning rate policies, and data augmentation strategies prevent the experimental setup from favoring any particular method.
🚫
No External Data
Strict prohibition of using additional or synthetic data beyond the original benchmark training set ensures observed gains stem solely from the learning algorithm.
📊
Dual-Metric Framework
Evaluates both robustness performance (accuracy against strong attacks) and computational cost (GPU hours and memory footprint).
🔬
Comprehensive Evaluation
Multiple attack strengths and evaluation protocols provide a complete picture of each method's robustness-accuracy trade-off.
🔄
Unified Implementation
All methods implemented in a single framework with consistent coding style, reducing implementation bias and improving reproducibility.
