AdvTransfer Benchmark

A Comprehensive Framework for Fair Evaluation of Adversarial Transferability Methods

📋 Overview

The AdvTransfer Benchmark provides a rigorous and fair evaluation framework for adversarial transferability methods. Unlike existing evaluations that use inconsistent model architectures and preprocessing pipelines, our benchmark establishes unified conditions where all attack methods compete on equal footing.

This platform implements representative transferability methods including NAA, COPIA, and their variants with a unified codebase, ensuring fair comparison across different algorithmic innovations. The benchmark systematically removes advantages from model ensemble and unlimited query budgets, providing a transparent baseline for evaluating black-box adversarial attacks.
